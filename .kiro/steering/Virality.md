# 1. Objective

Design a **self-reinforcing viral loop** where:

* Every user becomes a distribution node
* Every profile is a conversion funnel
* Every interaction drives token demand

---

# 2. Core Viral Thesis

> Users grow by promoting themselves — and others profit from it.

This leverages:

* Social status
* Financial upside
* Network effects

Virality emerges when:

* users share because it benefits them directly ([Medium][1])

---

# 3. Core Viral Loop

## Loop Definition

1. User creates profile

2. Gets subdomain:

   ```
   runnername.ontrail.tech
   ```

3. Shares profile:

   * X (Twitter)
   * Email
   * Direct link

4. New user lands → signs up (1-click via Privy)

5. New user:

   * auto-follows inviter
   * buys FriendPass
   * starts tipping / investing

6. Original user:

   * gains reputation
   * gains token demand
   * increases price

→ LOOP REPEATS
